Etapa 1: Entendendo o Problema#

Antes de treinar qualquer modelo, precisamos definir claramente:

1.1 Qual √© o Problema?#

Exemplos de Problemas:

  • Classifica√ß√£o: Email √© spam ou n√£o spam?
  • Regress√£o: Qual o pre√ßo deste im√≥vel?
  • Clustering: Quais clientes s√£o similares?
  • Detec√ß√£o de Anomalias: Esta transa√ß√£o √© fraude?

1.2 Tipo de Aprendizado#

Supervisionado:

  • Voc√™ tem dados com r√≥tulos/respostas corretas
  • Exemplo: Fotos de gatos e cachorros j√° classificadas

N√£o Supervisionado:

  • Voc√™ tem dados sem r√≥tulos
  • Exemplo: Segmentar clientes sem categorias pr√©-definidas

Por Refor√ßo:

  • Agente aprende por tentativa e erro

1.3 M√©tricas de Sucesso#

Como saberemos se o modelo √© bom?

Classifica√ß√£o:

  • Acur√°cia: % de previs√µes corretas

  • Recall: % de positivos encontrados
  • F1-Score: M√©dia harm√¥nica de precis√£o e recall

Regress√£o:

  • MAE (Mean Absolute Error): Erro m√©dio absoluto
  • RMSE (Root Mean Squared Error): Raiz do erro quadr√°tico m√©dio

8.2 API Simples com Flask#

from flask import Flask, request, jsonify
import joblib
import pandas as pd

app = Flask(__name__)

# Carregar modelo e scaler
modelo = joblib.load('modelo_precos_imoveis.pkl')
scaler = joblib.load('scaler.pkl')

@app.route('/prever', methods=['POST'])
def prever():
    # Receber dados JSON
    data = request.get_json()
    
    # Converter para DataFrame
    df_input = pd.DataFrame([data])
    
    # Normalizar
    df_scaled = scaler.transform(df_input)
    
    # Prever
    predicao = modelo.predict(df_scaled)[0]
    
    # Retornar resultado
    return jsonify({'preco_previsto': float(predicao)})

if __name__ == '__main__':
    app.run(debug=True, port=5000)
